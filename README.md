# Artificial Intelligence — Lab

Course: Artificial Intelligence - Lab
Program: BSCS / Computer Science
Duration: 16 weeks

## Overview

This repository contains my practice labs, graded assignments, and project files completed during the Artificial Intelligence Lab course. It is organized week-by-week to help learners follow the syllabus, reproduce search algorithms, and run machine learning models locally.

## Repository structure

* [**/practice-labs**](https://github.com/Malik-Faisal-Awan1/[Your-AI-Repo-Name]/tree/main/practice-labs) — weekly hands-on exercises (Practice Lab 01 to 14) with instructions and Python code
* [**/graded-labs**](https://github.com/Malik-Faisal-Awan1/[Your-AI-Repo-Name]/tree/main/graded-labs) — graded lab assignments (01 to 04) and my submissions
* [**/exams**](https://github.com/Malik-Faisal-Awan1/[Your-AI-Repo-Name]/tree/main/exams) — mid-term and final-term review materials and code
* [**/project**](https://github.com/Malik-Faisal-Awan1/[Your-AI-Repo-Name]/tree/main/project) — final project evaluation files and source code

## How to use

1. Clone the repo: `git clone https://github.com/Malik-Faisal-Awan1/[Your-AI-Repo-Name].git`
2. Create a virtual environment: `python -m venv venv`
3. Activate and install dependencies:
   * Windows: `venv\Scripts\activate`
   * macOS/Linux: `source venv/bin/activate`
   * `pip install -r requirements.txt`
4. Follow the README inside each specific weekly folder to run the practice labs.

## Highlights

* **Core topics covered:** Problem Formulation in Python, Uninformed Search (BFS, DFS, Uniform Cost Search), Informed Search (Greedy Best First, A*), Hill Climbing, Adversarial Search (MiniMax, Alpha-Beta Pruning), Propositional & First Order Logic, Machine Learning (Bayes Classifier, Clustering), and Deep Learning (Neural Networks).
* **Structure:**
    * **Weeks 1-3:** Python basics and Problem Formulation
    * **Weeks 4-7:** Uninformed, Informed, and Hill Climbing Searches
    * **Week 8:** Mid Term
    * **Weeks 9-10:** Adversarial Search and Genetic Problems
    * **Weeks 11-12:** Logic (Propositional and First Order)
    * **Weeks 13-15:** Machine Learning & Deep Learning concepts
    * **Week 16:** Final Term
